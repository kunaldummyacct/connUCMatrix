# Connector Use Case Matrix Utility

## Quick Start Guide

### For Windows Users:
1. Extract the zip file to a folder
2. Copy your `ConnUCMatrix.xlsx` file into the folder
3. Double-click `setup.bat` (first time only)
4. Double-click `run.bat` to start
5. Open browser to http://localhost:5000
6. Login with password: `admin123`

### For Mac Users:
1. Extract the zip file to a folder
2. Copy your `ConnUCMatrix.xlsx` file into the folder
3. Open Terminal and navigate to the folder
4. Run: `./setup.sh` (first time only)
5. Run: `./run.sh` to start
6. Open browser to http://localhost:5000
7. Login with password: `admin123`
